## How to Run

To run the smart contracts, you can use the following steps:
1. Install Dependencies: Make sure you have a Solidity development environment set up. You can use Remix IDE, Truffle, or Hardhat.
2. Compile: Compile the contracts using your development environment.
3. Deploy: Deploy the contracts on a local or test blockchain.
4. Interact: Interact with the deployed contracts using the available functions.
